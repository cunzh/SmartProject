智慧社区管理系统使用说明

系统的配置

/SmartCommunity/src/jdbc.property -- 配置数据库地址，用户名，密码

智慧社区管理系统接口使用说明

  • 公告管理
    1. 发布公告
      2. 地址 ： host:port//MyBatis/notice/publishNotice.action
      参数 ： title , content,author
      返回结果 ：
      success : 是否成功
      type : 失败原因
    2. 查询公告
      3. 地址 ： host:port//MyBatis/notice/listNoticeByPage.action
      参数 ： pageSize,pageNo
      返回结果：
      success : 是否成功
      result : 返回结果数组
      {
      author : 发布者
      content : 公告内容
      date ： 发布日期
      title : 公告标题
      id ： 公告标识
      }
      totalpage : 总页数
  • 报修管理
    1. 列出设备名称
      2. 地址：http://localhost:8080/MyBatis/repair/listDevices.action 参数：无 返回结果 { id devicename }
    2. 列出设备可能出现的故障
      3. 地址：http://localhost:8080/MyBatis/repair/listDeviceFault.action 参数：id 返回结果 { id deviceid devicestate }
